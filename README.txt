🎮 VALORANT MATCH PREDICTION SYSTEM 🎮
=====================================

📊 WHAT IS THIS?
================
A machine learning system that predicts Valorant match winners!
Uses team matchups and map data from EMEA, AMERICAS, and PACIFIC regions.

🚀 HOW IT WORKS
===============

📈 STEP 1: EXPLORE DATA (First Model.ipynb)
   • Open the Jupyter notebook
   • Look at team performance patterns
   • Test different AI models
   • See which model works best

🎯 STEP 2: MAKE PREDICTIONS (valorant_predictor.py)
   • Choose your region (EMEA/AMERICAS/PACIFIC)
   • Pick two teams to compare
   • Select a map
   • Get your prediction!
